Privacy Policy

Your privacy, and the security of any information you submit (e.g., via the contact form or via comments), is of the utmost importance, and all efforts are made to ensure your data remains secure.

What Is Collected

Information about you may be collected in the following ways:

  • Page Tracking
    • The pages you view on this site are recorded along with a randomly-generated identifier that is used to allow overall site use to be determined per visitor. For example, this allows determination that a single visitor visited page X, then page Y, then page Z, as opposed to three separate visitors visiting one of those pages each, or that a single visitor returned to page X one hundred times, as opposed to one hundred different visitors viewing that page. Understanding this can help improve the site by directing efforts where they will have greatest impact.
    • This identifier is stored in a browser/client-side cookie, and a new, independent identifier is generated if one is not present. This means that if, for example, you clear your cookies or use a private window, a new identifier will be used.
  • Page Comments
    • If you submit a message via the Comment form on on any page, any information you provide is collected.
    • The page-tracking identifier (above) is also implicitly included with submitted comment messages. Note that this means your message can then be associated with the pages you visit on the site. If you don’t wish this to occur, you can use a private browser window for submission, ensuring a separate, independent identifier is sent with the message.
  • Site Contact
    • If you submit a message via the Contact form, any information you provide is collected.
    • The page-tracking identifier (above) is also implicitly included with submitted contact messages. Note that this means your message can then be associated with the pages you visit on the site. If you don’t wish this to occur, you can use a private browser window for submission, ensuring a separate, independent identifier is sent with the message.

Where It Is Stored

Information that is collected is stored within Google services (specifically Google Forms and Google Sheets) and may be additionally stored in private GitHub repositories that contain the source material for this site.
